{# ===== WHAT IS DDOS ===== #}

Что такое DDoS-атака и почему нужна защита

DDoS-атака (Distributed Denial of Service) — это распределённая атака на отказ в обслуживании, при которой злоумышленники направляют огромный объём трафика на целевой сервер, сеть или приложение. Цель атаки — сделать ресурс недоступным для легитимных пользователей.

Современные DDoS-атаки достигают мощности в сотни гигабит и даже терабит в секунду. Без профессиональной защиты даже крупные провайдеры не способны противостоять таким объёмам. AntiDDoS.su обеспечивает фильтрацию до 10 Tbps входящего трафика.

Уровни защиты

Многоуровневая фильтрация для полной нейтрализации атак

L3

Сетевой уровень

Защита от объёмных атак: UDP flood, ICMP flood, IP fragmentation. Фильтрация на уровне пакетов с минимальной задержкой.

  • • UDP / ICMP Flood
  • • IP Fragmentation
  • • Amplification-атаки (DNS, NTP, memcached)

L4

Транспортный уровень

Защита TCP-соединений от SYN flood, ACK flood и других атак на состояние соединения. Stateful-инспекция трафика.

  • • TCP State Exhaustion
  • • Connection Flood
  • • Slowloris
L7

Прикладной уровень

Интеллектуальная фильтрация HTTP/HTTPS-запросов. Выявление и блокировка ботов, парсеров и application-layer атак.

  • • HTTP Flood
  • • Slow HTTP Attacks
  • • API Abuse
  • • Browser Challenge
{# ===== BGP SECTION ===== #}

Как работает BGP-защита

BGP (Border Gateway Protocol) — протокол динамической маршрутизации, используемый для обмена маршрутной информацией между автономными системами в интернете.

01

Анонсирование

Мы анонсируем ваши IP-префиксы через нашу автономную систему. Весь входящий трафик перенаправляется на наши центры очистки.

02

Очистка

Трафик проходит через многоуровневые фильтры. Вредоносные пакеты отбрасываются, легитимные — пропускаются.

03

Доставка

Очищенный трафик доставляется на ваш сервер через GRE-туннель или прямое подключение с минимальной задержкой.
